🍬 CandyOS

A Magisk module that replaces the default Android boot animation with the CandyOS boot animation.

«[!IMPORTANT]
This module only replaces the boot animation.
No system partitions are modified.»

---

✨ Features

- 📱 Easy installation through Magisk
- ⚡ Lightweight and optimized
- 🔄 Automatically removed after uninstalling the module
- 📦 Supports multiple boot animation paths
- 🛡️ Tested on HyperOS

---

📋 Requirements

- Android 12 or newer
- Magisk 30.0+
- Root access

---

📥 Installation

1. Download the latest release.
2. Open the Magisk app.
3. Go to Modules.
4. Tap Install from storage.
5. Select CandyOS.zip.
6. Reboot your device.

---

📂 Supported Paths

/system/media/bootanimation.zip
/system/product/media/bootanimation.zip
/product/media/bootanimation.zip
